# Customer GA Activation Checklist

Production activation is future-only and is not executed by R9.

- [ ] Business approval recorded.
- [ ] Deployment window approved.
- [ ] Backup confirmation completed.
- [ ] Server access confirmed for approved engineering staff.
- [ ] DB migration plan reviewed and separately approved.
- [ ] EDGE connectivity plan reviewed.
- [ ] LINK connectivity plan reviewed.
- [ ] Rollback approval authority confirmed.
- [ ] Acceptance signer identified.
- [ ] Production support window confirmed.
- [ ] Security/secrets handling plan confirmed; no secrets are bundled in R9.
- [ ] Runtime activation approval explicitly separated from delivery scaffold acceptance.
