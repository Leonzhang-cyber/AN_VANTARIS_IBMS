# IDC / DCIM A0 Capacity and Energy Model

## 1. Capacity Model

- rack U-space capacity
- rack power capacity
- rack weight capacity
- rack network port capacity
- room power capacity
- room cooling capacity
- UPS capacity
- PDU capacity
- redundancy level, future
- capacity reservation, future

## 2. Energy / Efficiency Model

- total IT load
- total facility load
- UPS load
- PDU load
- cooling load
- estimated PUE
- power usage by room/row/rack
- cooling efficiency, future
- thermal anomaly count
- energy impact to ESG

## 3. Risk / Resilience Model

- power chain risk
- cooling chain risk
- rack overload risk
- high temperature risk
- humidity risk
- single point of failure indicator, future
- alarm severity impact to availability

## 4. A0 Boundary

A0 defines capacity/energy/risk boundaries only.
It does not create a real DCIM calculation engine.
It does not create DB table.
It does not modify API.
It does not modify frontend.
