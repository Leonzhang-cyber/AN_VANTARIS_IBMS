# VANTARIS ONE Platform Metadata Policy

Rules:

- metadata files are governance-only
- metadata must not claim runtime readiness
- metadata must not imply GA
- package skeleton status must remain SKELETON_ONLY until source migration
- platform name is VANTARIS ONE
- IBMS remains ibms-core
- legacy AN_VANTARIS_IBMS remains source base until approved migration
- no API/DB/runtime rename based only on metadata
- version values in metadata are transition versions, not release versions
