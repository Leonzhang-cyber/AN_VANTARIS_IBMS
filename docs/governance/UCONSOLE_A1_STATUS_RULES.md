# UCONSOLE A1 Status Rules

1. UConsole is a platform status module draft, not runtime implementation.
2. UConsole does not implement frontend pages in A1.
3. UConsole does not create menu routes in A1.
4. UConsole does not implement backend APIs in A1.
5. UConsole does not create DB tables, schemas, or migrations.
6. UConsole does not implement dashboard runtime in A1.
7. UConsole does not implement auth/login/RBAC in A1.
8. UConsole does not own Edge or Link runtime.
9. UConsole does not own Contracts schemas.
10. UConsole does not include real secret/token/credential content.
11. Foundation dependencies are reference-only in status model drafts.
12. A1 output remains docs-only and non-executable.
