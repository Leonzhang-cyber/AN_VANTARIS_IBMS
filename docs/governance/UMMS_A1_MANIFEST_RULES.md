# UMMS A1 Manifest Rules

1. UMMS is a business module, not a Foundation module.
2. UMMS does not own Edge runtime.
3. UMMS does not own Link runtime.
4. UMMS does not own DB schema.
5. UMMS does not own Contracts schema.
6. UMMS does not own backend runtime.
7. UMMS does not own frontend runtime.
8. UMMS does not implement real work order engine in A1.
9. UMMS does not implement technician scheduling in A1.
10. UMMS does not add menu route/API/DB table in A1.
11. UMMS does not copy UFMS runtime/source/schema/auth/login/seed/migration.
12. UMMS does not include real credential/secret/token/private key content.
