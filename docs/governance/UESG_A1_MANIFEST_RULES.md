# UESG A1 Manifest Rules

1. UESG is a business module, not a Foundation module.
2. UESG does not own Edge runtime.
3. UESG does not own Link runtime.
4. UESG does not own DB schema.
5. UESG does not own Contracts schema.
6. UESG does not own backend runtime.
7. UESG does not own frontend runtime.
8. UESG does not implement ESG regulatory engine in A1.
9. UESG does not implement carbon accounting engine in A1.
10. UESG does not add menu route/API/DB table in A1.
11. UESG does not copy UFMS runtime/source/schema/auth/login/seed/migration.
12. UESG does not include real credential/secret/token/private key content.
