# UCORE A1 Manifest Rules

1. UCore is a business module, not a Foundation module.
2. UCore does not own Edge runtime.
3. UCore does not own Link runtime.
4. UCore does not own DB schema.
5. UCore does not own Contracts schema.
6. UCore does not own backend runtime.
7. UCore does not own frontend runtime.
8. UCore does not implement command center runtime in A1.
9. UCore does not implement auth/login/RBAC runtime in A1.
10. UCore does not add menu route/API/DB table in A1.
11. UCore does not copy UFMS runtime/source/schema/auth/login/seed/migration.
12. UCore does not include real credential/secret/token/private key content.
