# REPORTS A3 Catalog Governance Rules

1. Reports catalog in A3 is docs-level only.
2. No report engine implementation is allowed in A3.
3. No backend API implementation is allowed in A3.
4. No DB implementation is allowed in A3.
5. No frontend implementation is allowed in A3.
6. No export runtime implementation is allowed in A3.
7. No scheduled job implementation is allowed in A3.
8. Report catalog candidates must not expose sensitive evidence or UFMS fault details without future RBAC/security review.
9. Export and schedule capabilities require separate explicit authorization.
10. Future implementation must pass REPORTS-A4 implementation gate.
