# REPORTS A1 Manifest Rules

1. Reports is a platform module.
2. Reports does not own external system ingress.
3. Reports does not own EDGE runtime.
4. Reports does not own LINK runtime.
5. Reports does not own Contracts schema.
6. Reports does not own DB schema.
7. Reports does not copy UFMS/UMMS/UESG/UCDE/UDOC runtime.
8. Reports does not introduce credentials or secrets.
9. Reports does not add menu route in A1.
10. Reports does not add API in A1.
11. Reports does not add DB table in A1.
12. Reports A2 may propose API/data model gate and still requires separate authorization.
