# UDOC A1 Manifest Rules

1. UDOC is a business module, not a Foundation layer module.
2. UDOC does not own Edge runtime.
3. UDOC does not own Link runtime.
4. UDOC does not own DB schema.
5. UDOC does not own Contracts schema.
6. UDOC does not copy or import UFMS runtime.
7. UDOC does not copy MMS/ESG/CDE runtime.
8. UDOC does not introduce credentials or secrets.
9. UDOC does not add menu route in A1.
10. UDOC does not add API in A1.
11. UDOC does not add DB table in A1.
12. Only A2 may propose contract draft, and still requires separate authorization.
