# EDGE Generated Consumers

This path is for generated consumer artifacts only.

Allowed generated consumer paths:

- `generated/contracts`
- `generated/protocol`
- `generated/security`

Forbidden:

- importing LINK runtime package code
- importing DB/Prisma code
- importing Console or NexusAI code
- secrets or `.env` usage

Authority for contracts remains `AN_VANTARIS_Contracts`.
