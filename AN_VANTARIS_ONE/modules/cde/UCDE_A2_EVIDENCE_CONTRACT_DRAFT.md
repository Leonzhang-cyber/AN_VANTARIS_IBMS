# UCDE A2 Evidence Contract Draft

UCDE is the primary name for this module.
CDE is retained only as historical A1 wording.
The `modules/cde` location is a historical path retained to avoid out-of-scope directory movement in this task.

UCDE-A2 is a VANTARIS ONE docs-level evidence contract draft only. It does not modify AN_VANTARIS_Contracts, contracts/, schemas/, backend/, frontend/, DB, Edge, or Link.

## 1. Scope

- docs-level evidence contract draft only
- not a real AN_VANTARIS_Contracts contract
- no changes to `contracts/`
- no changes to `schemas/`
- no OpenAPI creation
- no JSON Schema creation
- no backend/frontend/DB/runtime implementation

## 2. Draft Semantics Defined in A2

UCDE A2 defines draft semantics for:

- evidence identity
- evidence context
- evidence integrity references
- cross-module linkage references
- retention and classification references
- redaction and privacy references
