# UESG Module Manifest Draft

## 1. Purpose

This file is the UESG A1 module manifest draft.
UESG is the primary name. ESG, esg, Energy & Sustainability, and Sustainability Operations are historical wordings.

## 2. Scope

- docs-only
- manifest draft only
- governance-only alignment
- no runtime/API/DB/frontend implementation

## 3. Ownership Boundary

UESG owns sustainability and energy interpretation context references and linkage references across UDOC, UMMS, UCDE, and UCore.

UESG does not own Edge/Link/DB/Contracts runtime or schemas, and does not implement regulatory/carbon engines in A1.

## 4. Foundation Relationship

Foundation Layer is reference-only in this task. UESG consumes shared foundation references through approved boundaries only.

## 5. A1 Boundary

UESG A1 does not create runtime, API, DB schema, menu route, or frontend page.
